Bedoeling zo weinig mogelijk punten halen. Elke slag is 1 punt. Indien 7 slagen komt er een strafpunt bij.

**Oefening:**
```
begin programma
  lees slagen = invoer aantal slagen.
  lees punten = 0
  
  if slagen > 7
    then druk slagen + 1
  else if slagen < 7
    var slagen + 1
    
    druk slagen.
  endif

eind programma
```

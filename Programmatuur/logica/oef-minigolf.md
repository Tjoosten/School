**Oefening:**
```
begin programma
  lees slagen = invoer aantal slagen.
  lees punten = 0
  
  if slagen > 7
    then druk slagen + 1
  else if slagen < 7
    var punten + 1
    
    druk slagen.
  else
    druk uw hebt 8 punten voor deze baan.
  endif

eind programma
```
